# Локальный прототип публичного сайта

## Цель

Первый прототип объясняет назначение будущего сайта аналитики клана, показывает планируемые разделы, структуру состава и публичные правила обычных КВ. Он предназначен для локальной проверки содержания, визуального направления, доступности и адаптивности до подключения данных.

Прототип не является официальным сайтом Clash of Clans или Supercell.

## Реализованные блоки

- верхняя навигация и статус прототипа;
- первый экран с назначением проекта;
- карточки будущих модулей;
- пустое состояние будущей таблицы состава;
- краткое объяснение политики обычных КВ;
- этапы развития;
- принципы приватности;
- footer со статусом данных и версией прототипа.

Реальные участники, теги, должности, уровни ратуш и игровые показатели не добавлены. Состав появится только после подключения подтверждённого источника и согласования публичных полей.

## Технические границы

Сайт использует только HTML5, CSS и небольшой vanilla JavaScript для мобильного меню. Внешние зависимости, CDN, аналитика, cookies, `localStorage`, API-вызовы и сетевые ресурсы отсутствуют. Основное содержание и навигация доступны при отключённом JavaScript.

Внутренние заметки руководства не относятся к публичному сайту, не копируются в HTML и не входят в публичные ресурсы.

site/

## Локальный просмотр

Файл `site/index.html` можно открыть напрямую в браузере. Для проверки поведения через HTTP, если Python уже установлен, допустим временный локальный сервер:

```powershell
python -m http.server 8000 --bind 127.0.0.1 --directory D:\coc\repo\site
```

Сервер нужен только для локального просмотра. После проверки процесс следует завершить. Порт не должен публиковаться во внешнюю сеть.

## Следующие данные

Следующий содержательный этап — состав клана и базовые карточки игроков. До реализации нужно определить подтверждённый источник состава и утвердить разрешённые публичные поля карточки. Подключение API, локального хранилища и публикация сайта являются отдельными будущими этапами.
