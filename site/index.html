Понятные правила без длинного регламента

Как будет работать статистика КВ

Показатели должны объяснять участие, а не автоматически оценивать человека. Спорные случаи остаются предметом отдельного решения.

Политика прототипа
  1. 01

    Первая атака — по зеркалу

    Это ожидаемая цель по умолчанию. Возможны согласованные тактические исключения.

  2. 02

    Ненужная атака — не пропуск

    Если полезных целей не осталось, вторая атака получает нейтральный статус not_needed.

  3. 03

    Проверка по двум КВ

    После двух обычных войн без спорных ситуаций игрок может получить статус проверенного.

  4. 04

    Обычные КВ — примерно 50/50

    Состав сочетает проверенных и проверяемых игроков. В ЛВК используется больше проверенных.
