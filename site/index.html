О проекте

Прозрачный интерфейс с приватностью по умолчанию

Clan Analytics показывает только разрешённую публичную проекцию: ники, роли, ратуши и нейтральные агрегаты. Теги игроков, токен и внутренние заметки в публичную версию сайта не попадают.

01

Только разрешённые поля

Публикуются понятные поля без персональных и служебных идентификаторов.

02

Нет выдуманных результатов

Отсутствующая военная история не отображается как нулевой результат игрока.

03

Публичная версия без служебных данных

Сайт показывает разрешённые игровые сведения и не публикует токен, player tags или внутренние заметки.
